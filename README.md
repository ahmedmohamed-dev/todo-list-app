# To-Do List App

A simple task management application built with HTML, CSS, and JavaScript.

## 🚀 Live Demo
https://ahmedmohamed-dev.github.io/todo-list-app/

## ✨ Features
- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Save tasks using LocalStorage

## 🛠 Technologies
- HTML5
- CSS3
- JavaScript
- LocalStorage
